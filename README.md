# Sentiment Analysis of Twitter Data using NLP (Indian Election Dataset)
About the Project

In this project, I worked on analyzing Twitter data related to the Indian Elections to understand public sentiment. The goal was to classify tweets into positive, negative, or neutral and extract meaningful insights from real-world text data.

This project helped me understand how raw text data can be cleaned, processed, and transformed into useful information using NLP techniques.

 Tools & Technologies
Python
Pandas
NLTK
Scikit-learn
TF-IDF
Matplotlib
 What I Did
Cleaned raw tweet data by removing noise like special characters, emojis, and extra spaces
Converted text into a structured format using preprocessing techniques
Applied tokenization, stopword removal, and lemmatization
Converted text into numerical features using TF-IDF
Performed sentiment analysis to classify tweets
Analyzed the results to find patterns and insights
 Key Findings
Most of the tweets showed a positive sentiment
Some users had consistently higher positive engagement
The dataset revealed clear patterns in public opinion
Text data can provide valuable insights when processed correctly
