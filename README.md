# 📊 Sentiment Analysis of Twitter Data using NLP (Indian Election Dataset)

🚀 This project analyzes Twitter data to understand public sentiment using NLP techniques and extracts meaningful insights from real-world data.

---

## 📌 About the Project

In this project, I worked on analyzing Twitter data related to the Indian Elections to understand public opinion. The goal was to classify tweets into **positive**, **negative**, and **neutral** sentiments and extract meaningful insights from the dataset.

---

## 🔧 Tools & Technologies

* Python
* Pandas
* NLTK
* Scikit-learn
* TF-IDF
* Matplotlib

---

## ⭐ Project Highlights

* Built an end-to-end NLP pipeline
* Cleaned and processed real-world text data
* Applied text preprocessing techniques
* Performed sentiment classification
* Generated insights from unstructured data

---

## 🧠 Workflow

* Data Cleaning (removing noise, special characters, emojis)
* Text Preprocessing (tokenization, stopword removal, lemmatization)
* Feature Extraction using TF-IDF
* Sentiment Analysis
* Data Analysis and visualization
